pub fn desired_requests( &self ) -> impl Iterator<Item = (SourceId, &TSrc, DesiredRequest)> + '_

Returns a list of requests that should be started in order to drive the warp syncing process to completion.

Once a request that matches a desired request is added through InProgressWarpSync::add_request, it is no longer returned by this function.
